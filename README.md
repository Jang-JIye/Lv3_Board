  - 부모 테이블인 게시글 (board)에서 댓글이 없을 경우, 삭제가 안되는 문제점이 있었습니다.

7. 5번과 같은 문제가 발생했을 때 JPA에서는 어떻게 해결할 수 있을까요?
 - 부모 엔티티에 게시글 인스턴스에 `@OneToMany(... cascade = CascadeType.ALL)` 추가
 - 자식 엔티티 모두 삭제 후 부모 엔티티 삭제 

8. IoC / DI 에 대해 간략하게 설명해 주세요!
